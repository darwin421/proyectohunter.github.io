ISAC // Módulo de Amenazas

Señal establecida… parcial. Prioridad: identificar niveles de peligro y comportamiento hostil.

La red SHD clasifica a los enemigos según su nivel de amenaza. Cada categoría utiliza un código de color para facilitar su identificación.

Clasificación de Hostiles por Color

  • Rojo — Hostil Estándar
    Salud baja. Sin armadura. Eliminación rápida.
  • Morado — Hostil Veterano
    Salud y armadura superiores. No presentan regeneración.
  • Dorado — Hostil Élite
    Armadura pesada. Salud incrementada. Capacidades especiales detectadas.
  • Dorado con Identificador — Hostil con Nombre
    Unidad prioritaria. Armadura reforzada. Salud muy elevada. Capacidades avanzadas.

ISAC // Nota
El color indica nivel de peligro, no el tipo de arma que portan.

Facciones Enemigas

ISAC // Registro
Los hostiles detectados pertenecen a múltiples facciones enemigas. No existe cooperación entre ellas. Conflictos internos registrados. Controlan zonas específicas del mapa.

Cada facción presenta tácticas, armamento y comportamiento distintos. Información detallada disponible en el módulo del Agente.

Comportamiento General de Facciones

ISAC // Análisis
Métodos de combate detectados:

  • Uso de fuego y armas automáticas.
  • Uso de armas cuerpo a cuerpo combinadas con potenciadores químicos.
  • Uso de armamento improvisado.
  • Uso de tecnología modificada.

Advertencia: tácticas específicas serán detalladas en el módulo del Agente.

Interacción entre Facciones

ISAC // Informe
No existe alianza entre facciones enemigas. Interacciones hostiles registradas entre ellas.

Conclusión: el usuario puede aprovechar conflictos entre facciones para reducir riesgo.

Renegados

ISAC // Registro
Se han identificado agentes de la División que han abandonado el protocolo SHD. Estado: hostiles. Clasificación: facción independiente. Nivel de amenaza: extremadamente alto.

Nota: tácticas, armamento y comportamiento serán detallados en el módulo del Agente.

Hostil Desconocido Altamente Blindado

ISAC // Error de Identificación
Señal detectada en múltiples ocasiones. Clasificación automática: desconocido. Blindaje: superior a cualquier facción registrada.

Observación: presencia detectada en zonas elevadas y tejados. Comportamiento: vigilancia pasiva del usuario.

Estado del archivo: corrupto. Información adicional no disponible en red local.

Unidades Hostiles con Habilidades Anómalas

ISAC // Advertencia

Estado: información incompleta. Acceso total requiere conexión a la red SHD.
