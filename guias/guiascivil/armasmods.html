Capacidad de Modificación

ISAC // Registro
El número máximo de modificadores por arma es cuatro. Este valor varía según el tipo de arma.

  • Pistolas: 1–2 modificadores.
  • Armas compactas: 2–3 modificadores.
  • Armas estándar: hasta 4 modificadores.

Algunas armas incluyen modificadores integrados de fábrica. Estado: modificación bloqueada.

Modificación de Cargadores

ISAC // Análisis
Los cargadores modifican:

  • Capacidad total de munición.
  • Velocidad de recarga.

Advertencia: cargadores de alta capacidad reducen velocidad de recarga.

Modificación de Ópticas

ISAC // Análisis
Las ópticas incrementan:

  • Precisión.
  • Control del disparo.
  • Capacidad de apuntar a distancia.

Advertencia: algunas ópticas reducen velocidad de apuntado.

Modificación de Boca de Arma

ISAC // Análisis
Los accesorios de boca modifican:

  • Retroceso.
  • Dispersión del disparo.
  • Estabilidad general.

Advertencia: algunos modelos reducen distancia óptima.

Modificación de Empuñaduras

ISAC // Análisis
Las empuñaduras mejoran:

  • Estabilidad.
  • Manejo del arma.
  • Control del retroceso.

Advertencia: algunas reducen precisión a larga distancia.

Modificadores con Efectos Negativos

ISAC // Advertencia
No todos los modificadores son mejoras absolutas. Algunos incrementan un atributo mientras reducen otro.

  • Mayor capacidad → menor velocidad de recarga.
  • Mayor estabilidad → menor distancia óptima.
  • Mayor precisión → reducción en manejo.
  • Mayor control de retroceso → menor cadencia efectiva.

Datos Dañados // Exóticas

Archivo corrupto detectado… Información parcial disponible.

Las armas exóticas incluyen modificadores únicos no reemplazables. Estos accesorios forman parte del diseño del arma.

Datos adicionales requieren acceso a la red SHD.
